Check the weather on iPhone

Use this Weather app to check the weather for any location. You can also view the upcoming forecast, see information about severe weather conditions and, if rain or snow is on the way.

Home Page

  • Home page will show a list of locations that the user has bookmarked previously.
  • On click of delete button present on the right side corner, you can remove locations from the list.
  • If you want to add location , click on '+' button at the top right corner.

Add Locations

On single tap over map you can select any location for which you want to check weather conditions. Now once you pick one location then click on "Add Location". This will add this location to bookmarked locations.
